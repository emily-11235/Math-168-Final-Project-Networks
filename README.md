# Math-168-Final-Project-Networks
## Organization
This GitHub has a notebooks folder, which contains the three notebooks for our code. This includes the lead-lag simulation code, the window simulation code, and the world events code. In the main repository, the data file that was used for this project is uploaded.

## How to Replicate
To replicate this code download each of the notebooks, and upload it to Google Colab. Then download the dataset. Upload it in the files section (to the lefthand side) of the Google Colab. Run each of the cells as needed.
